Le jeu de Hex est un jeu de stratégie abstrait pour deux joueurs, où le joueur 1 utilise la couleur bleue et le joueur 2 utilise la couleur rouge.

Le but du jeu est de créer une chaîne continue d'hexagones de sa propre couleur reliant deux bords opposés du plateau : le haut et le bas pour le joueur rouge, et la gauche et la droite pour le joueur bleu.

Le jeu se déroule sur un plateau hexagonal de taille variable, où les joueurs placent alternativement des hexagones de leur couleur sur les cases du plateau. Le joueur bleu commence en plaçant un hexagone bleu sur n'importe quelle case vide du plateau. Les joueurs se relaient ensuite pour placer un hexagone de leur couleur sur une case libre du plateau, et une fois placée, un hexagone ne peut plus être déplacée ou retirée du plateau.

La partie se termine dès qu'un joueur réussit à relier ses deux bords avec une chaîne continue de pierres de sa couleur. Le premier joueur à atteindre cet objectif est déclaré vainqueur.
